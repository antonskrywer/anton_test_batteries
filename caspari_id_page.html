Ihr individueller Code

Bitte erstellen Sie Ihren individuellen Code mit insgesamt acht Zeichen. Der Code setzt sich aus vier Teilen zusammen. Tragen Sie die Zeichen in genau dieser Reihenfolge ein: 

•	Die zwei Ziffern der Schule (siehe allgemeine Teilnehmendeninformation)		    _ _
Beispiel 00	→	00
•	Der erste und letzte Buchstabe des Vornamens Ihres Kindes			    _ _
Beispiel Martin	→	MN
•	Die beiden letzten Buchstaben des Nachnamens Ihres Kindes			    _ _
Beispiel Müller	→	ER
•	Der Monat, in dem Ihr Kind geboren wurde (zweistellig)				    _ _
Beispiel Mai	→	05
Der vollständige Code (Beispiel)	    00MNER05

Der vollständige Code (Beispiel): 00MNER05

Bitte achten Sie darauf, alle Angaben korrekt einzutragen. Der Code dient der anonymen Zuordnung der Daten.
